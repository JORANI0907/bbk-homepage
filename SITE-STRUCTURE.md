## 0. 디자인 시스템 (신규 추가)

### 0-1. 컬러 팔레트 (Industrial Mono + Vivid Pink)

| 역할        | 컬러             | 코드                       | 사용처                                                 |
| ----------- | ---------------- | -------------------------- | ------------------------------------------------------ |
| Primary     | Industrial Black | `#0B0B0F`                | 메인 배경(Hero 등 어두운 섹션), 본문 텍스트(밝은 섹션) |
| Surface BG  | Cool White       | `#F6F6F4`                | 밝은 섹션 배경 (Bento, 비교, 폼)                       |
| Surface 2   | Concrete Gray    | `#E5E5E2`                | 카드, 분리선, 보더                                     |
| Accent      | Vivid Pink       | `#FF2E63`                | 거대 워드마크, 메인 CTA, 강조 (단 1개 액센트로 통일)   |
| Accent Soft | Pink Tint        | `#FFD8E2`                | 핑크 카드 배경, 호버 글로우                            |
| Ink Inverse | Pure White       | `#FFFFFF`                | 어두운 섹션의 헤드/본문                                |
| Mute Light  | Stone 400        | `#8E8E8A`                | 어두운 섹션 보조 텍스트                                |
| Mute Dark   | Stone 600        | `#5A5A56`                | 밝은 섹션 보조 텍스트                                  |
| Hairline    | Hairline 10%     | `rgba(255,255,255,0.12)` | 어두운 섹션 분리선, 모노 라벨                          |

> **컬러 사용 원칙**: 액센트는 **Vivid Pink 단 1색만** 사용 (그라디언트·다중 액센트 금지). 어두운 섹션과 밝은 섹션을 교차 배치해 리듬을 만든다.
> **그라디언트**: 사용 안 함. 단색면 + 사진 + 핑크 워드마크의 3 레이어 구조로 통일.

### 0-2. 타이포그래피 (영/한 듀얼 시스템)

> Editorial Brutalism의 핵심: **영문 거대 워드마크 + 한글 헤드 + 모노스페이스 메타**의 3중 위계

| 역할              | 폰트                                  | 크기 (Desktop / Mobile) | 용도                                          |
| ----------------- | ------------------------------------- | ----------------------- | --------------------------------------------- |
| Wordmark XXL (영) | Neue Haas Grotesk Black / Inter Black | 320~480px / 140px       | Hero 거대 영문 워드마크 (CLEAN/NIGHT/ZERO 등) |
| Display XL (한)   | Pretendard Variable 900               | 88px / 44px             | Hero 한글 헤드 ("영업이 끝난 공간을…")       |
| Display L (한)    | Pretendard Variable 800               | 72px / 40px             | 섹션 헤드                                     |
| Headline (한)     | Pretendard Variable 700               | 48px / 28px             | 카드 헤드, 서브섹션                           |
| Body L            | Pretendard 500                        | 18px / 16px             | 도입 본문                                     |
| Body              | Pretendard 400                        | 16px / 15px             | 본문                                          |
| Label Mono        | JetBrains Mono / IBM Plex Mono 500    | 11~13px UPPERCASE       | 메타 라벨 (COMPANY REG, CAPACITY 등)          |
| Vertical Tag      | Mono 500 (rotated 90deg)              | 11px                    | Hero 우측 세로 텍스트                         |

> **타이포 원칙**
>
> - 영문 워드마크는 `tracking-[-0.04em]` + `leading-[0.85]` + `font-black` — 사진 위에 압도적으로 깔린다
> - 한글 헤드는 `tracking-[-0.02em]` + `leading-[1.05]` — 반점·마침표까지 카피 그대로 표기
> - 모노 라벨은 항상 `UPPERCASE` + `tracking-[0.08em]` — 산업 매뉴얼 톤
> - 본문은 `leading-relaxed (1.65)` 가독성 우선

### 0-3. 그리드 & 간격

- 12 컬럼, gap 24px (데스크탑) / 4 컬럼, gap 16px (모바일)
- 섹션 수직 패딩: `py-32` (데스크탑) / `py-20` (모바일)
- 컨테이너 최대 너비: `max-w-[1440px]` + 좌우 `px-6 md:px-12`

### 0-4. 모션 원칙 (Industrial Restraint)

> 부드러운 글래스 모션 → **차갑고 절제된 산업 모션**으로 톤 전환

| 트리거            | 효과                                                  | 사용처                 |
| ----------------- | ----------------------------------------------------- | ---------------------- |
| Page Load (Hero)  | 영문 워드마크 좌→우 wipe 0.9s + 한글 헤드 fade-up    | Hero 진입              |
| Scroll 진입       | `fade-up 16px + 0.08s stagger` (이지: easeOutQuart) | 모든 섹션 첫 진입      |
| Hover (Desktop)   | 이미지 살짝 zoom-in 1.04 + 핑크 보더 0→2px           | 사례 카드, 서비스 카드 |
| Cursor follow     | 핑크 점 커서 (mix-blend-difference)                   | 어두운 Hero 영역 한정  |
| Marquee           | `animate-scroll 40s linear infinite` (느리게)       | 업종 띠, 케어 품목 띠  |
| Sticky Reveal     | 스크롤 위치에 따라 우측 비주얼 cross-fade             | 서비스 2축 섹션        |
| Number Counter    | `easeOutExpo` 1.6s                                  | Stats Bar              |
| Magnetic Button   | 마우스 따라 ±6px 이동 + 보더 발광                    | 메인 CTA 버튼만        |
| Wordmark Parallax | 스크롤 시 영문 워드마크가 사진과 다른 속도로 이동     | Hero 깊이감 표현       |

> **금지**: 자동 재생 캐러셀, 스플래시 인트로, 타이프라이터, 무지개 그라디언트, 둥근 모서리 남용 (`rounded-full` 외 대부분 `rounded-none` 또는 `rounded-sm`)
> **기본 형태**: 사각형 — Editorial Brutalism은 직각의 미학

### 0-5. 컴포넌트 라이브러리 (Editorial Brutalism)

- **Wordmark Overlay** *(신규·핵심)*: 풀블리드 사진 위에 화면 폭을 넘는 영문 워드마크. 사진과 워드마크 사이 `mix-blend-mode` 또는 `object-fit-cover` + 텍스트 마스크
- **Asymmetric Hero Grid**: 좌상단 로고 / 우상단 네비 / 우측 세로 메타 / 우하단 메타 그리드 / 좌하단 CTA — 4 코너 활용
- **Mono Meta Grid**: 모노스페이스 라벨+값 4~6칸 그리드 (산업 사양표 느낌)
- **Outline Pill Badge**: `border + transparent bg` 외곽선 배지 (글래스 카드 대체)
- **Bento Grid**: 가변 크기 카드 (12-col span)
- **Sticky Scroll Reveal**: 좌 텍스트 / 우 비주얼 교체
- **Before/After Slider**: drag 비교
- **Marquee Ticker**: 느린 무한 스크롤
- **Number Counter**: 스크롤 진입 카운트업
- **Sectional Divider**: 섹션 사이 1px Hairline + 좌측 모노 라벨 (`SECTION 03 — SERVICES`)
- **Tag Cloud**: 외곽선 칩 (배지 대체)

  - 후보: `CLEAN` (시안과 동일·안정적) / `NIGHT` (야간 컨셉 강조) / `ZERO` (오염 제로) / `SHIFT` (마감 후 교대)

  - Primary: `서비스 알아보기` — 핑크 배경 / 흰 텍스트 / 사각 / 호버 시 핑크 발광

  - 스타일: `border border-white/60 text-white`, 모서리 사각, 작은 글씨, 핑크 도트(•) 좌측 prefix

## 4. 모바일 반응형 기준

## 5. 접근성 (WCAG 2.2 AA)

- 모든 모션은 `prefers-reduced-motion` 존중
- 거대 타이포라도 본문 컨트라스트 4.5:1 이상
- 폼 필드: 명확한 라벨 + 에러 메시지
- 키보드 포커스: 핑크 외곽선 (`ring-2 ring-[#FF2E63] ring-offset-2`)
- Before/After 슬라이더: 키보드 화살표로 조작 가능
- 영상 자동재생은 음소거 + 정지 버튼 제공
